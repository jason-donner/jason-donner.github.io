STACK: TABLEAU + MAPBOX + SET ACTIONS

The Design Challenge

Personal data is intimate, but often small in volume. With only 30 data points for my 2025 concert tour, a standard map felt sparse and utilitarian. My challenge was to transform a simple list of events into an immersive experience; something that felt substantial and worth exploring, rather than just a scattering of dots.

The Technical Architecture

I treated this dashboard like a software product, not a report. I used Mapbox Studio to build a custom 'Dark Glass' layer, removing 90% of the map detail to create a high-contrast stage for the data. For interactivity, I used Tableau Set Actions to create a fluid drill-down: clicking a single venue instantly isolates that specific memory in the side panel, creating a responsive, app-like feel without clunky filters.
